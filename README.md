# ?? Sistema de Gestión Veterinaria - PetSalud

Sistema completo para la gestión de exámenes y tratamientos veterinarios.

## ?? Características

- ? Registro de dueños y mascotas
- ? Gestión de órdenes veterinarias
- ? Registro de toma de muestras
- ? Procesamiento y validación de resultados
- ? Generación de informes PDF
- ? Sistema de facturación
- ? Reportes y KPIs
- ? Notificaciones automáticas

## ??? Tecnologías

**Backend:**
- Java 17
- Spring Boot 3.2
- MySQL
- Maven

**Frontend:**
- Angular 17
- TypeScript
- Bootstrap/Tailwind
